默写吧 1. 基于分治的快速排序思路： 每次取list 的第一个元素，把剩下的元素与第一个元素比较，此时生成两个list,less中存放的就是余下的所有比第一个元素小的list, greater 中存放的是余下的所有比第一个元素大的list。然后循环递归，当基线条件只有一个元素或0个元素时停止，并返回结果。 Python:123456789101112def quick_sort(arr): if len(arr)<2: return arr else: pivot = arr[0] less = [i for i in arr[1:] if i<=pivot] greater = [i for i in arr[1:] if i>pivot] return quick_sort(less) + pivot + quick_sort(greater)if __name__ == "__main__": arr = [5,7,9,3,6,1,4,2] print(quick_sort(arr))
